VKConnectorDelegate Protocol Reference

Conforms to NSObject
Declared in VKConnector.h

Overview

Протокол объявляет методы отслеживания изменения статуса токена доступа хранимого классом VKConnector.

Tasks

Modal view

  • – VKConnector:willShowModalView:

    Метод вызывается до того, как произойдет отображение модального окна авторизации
  • – VKConnector:willHideModalView:

    Метод вызывается до того, как произойдет скрытие модального окна авторизации

Access token

  • – VKConnector:accessTokenInvalidated:

    Метод, вызов которого сигнализирует о том, что токен стал недействительным, срок его действия истёк.
  • – VKConnector:accessTokenRenewalSucceeded:

    Метод, вызов которого сигнализирует о том, что токен доступа успешно обновлён.
  • – VKConnector:accessTokenRenewalFailed:

    Метод, вызов которого сигнализирует о том, что обновление токена доступа не удалось. Причиной ошибки может быть прерывание связи, либо пользователь отказался авторизовывать приложение.

  • – VKConnector:parsingErrorOccured:

    Метод, вызов которого сигнализирует о том, что произошла ошибка при парсинге JSON ответа сервера
