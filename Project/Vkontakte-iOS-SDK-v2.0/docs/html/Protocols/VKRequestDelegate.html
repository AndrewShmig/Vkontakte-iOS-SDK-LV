VKRequestDelegate Protocol Reference

Conforms to NSObject
Declared in VKRequest.h

Overview

Протокол инкапсулирует в себе основные методы для отслеживания состояния осуществляемого запроса.

Обязательным является лишь один метод - метод, который возвращает ответ, остальные являются опциональными.
